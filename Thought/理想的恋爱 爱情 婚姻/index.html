# 序

自大学期间就开始思考该问题 但是于现在才将其成文 本文内容仅代表个人观点 且基于被称之为人（普通人）和一定的经济基础（能够较为轻松地养活自己且）之上 并且能够独立自主生活 若不符合以上条件 本文不建议阅读

中国的恋爱进化史是悠远漫长的 从父母之命媒妁之言 再到现在的自由恋爱 传统的恋爱观与近代的恋爱观 这边不做详细阐述 传统的恋爱观有着气优秀的传统美德 近代恋爱观也有其糟粕 本文取其精华去其糟粕

由于本人语法功底差 故不添加断句标点符号 本文持续修正更新

关于赡养部分接受反驳 其余部分只接受讨论

# 概念

# 喜欢

喜欢是一种感觉 感觉是主观的 也基于客观之上 此处的客观指的是缘

某一次的拐角相撞 回眸一笑 夕阳下的转身 欢喜同桌 可以是一见钟情 也可以自日久生情

喜欢是想要认识一个人 了解一个人 从陌生 到朋友 为此辗转反侧 夜不能寐

# 爱

爱可以用三个词概括 责任 克制 恩爱 性生活 下面一一介绍
责任：

为自己负责 遵从本心 爱自己 且不能欺骗 背叛

为对方负责 关心对方 在乎对方

为两个人的生活负责 是个复杂的问题 生活离不开柴米油盐 一起生活的点点滴滴双方都应该付出努力

克制：

是想伸出去 却又收回去的手

是尊重 虽然已经是很亲密的关系 但是这一点依旧非常重要 但是此时的尊重是不同与朋友等其他关系之间 由于个体的差异 必然存在许多观点 习惯 兴趣等的不同 应当互相尊重

恩爱：

这个词在某个地方评论区看到 中国人讲恩爱 夫妻恩恩爱爱 你若许我一生 所爱皆我 我应你一世 双方接受了不求回报的付出 受之有恩 这份恩情 双方都为之付出一生 这也是白头偕老最重要的的一点

看到这个词的时候 想到了《红楼梦》绛珠草为报警幻仙子的甘露之恩 用一生的泪水偿还

原文：那绛珠仙子道：他是甘之惠，我并无此水可还。他若下世为人，我也去下世为人，但把我一生所有的眼泪还他，也偿还得过了。

性生活：

对于相爱的两个人来说这一点是必不可少且非常重要的 基于爱的性不是单纯的性行为 不需要多高尚 也不需要伪装 只是最原始 最真实的个人 在这里不得不提一下性癖 每个人都有自己的性癖 或许普通 或许难以接受 请做好沟通 与尊重

# 阶段

# 恋爱

仅讨论两情相悦

恋爱阶段是始于喜欢 从认识到相知 这个阶段最重要的就是交流 三观 性格

在这里自由恋爱就起到了很重要的作用 在此交流阶段若是存在不可调和的矛盾就应当分手 伤心痛苦都是必然的 但是依旧应该舍弃沉没成本 拥抱新人

在此阶段是拒绝性关系的 具有一定的身体接触距离 毕竟若不是未来的另一半 都不应越界 这是对自己负责 也是对对方负责

# 爱情

恋爱之上 喜欢之上 所谓爱情 便是爱

上文已详细解释了爱 此处不做重复

爱情是以结婚为前提 却不以结婚为目的 彼此认定相伴一生 不为一纸婚书而携手

# 婚姻

理想中 婚姻是双方独立出原先的家庭组建属于自己的家庭 不说女方嫁给男方或者其他方式 但是现实总是事与愿违 一个家庭需要经济支撑 且存在复杂的社会因素与法律因素

在旧时期 男方主要是经济收入的来源 承担家庭的开支 抚养 赡养 由于经济基础 女性一般嫁给男方 一起支撑一个家庭 由于科技的落后的 主要收入来源来自体力劳动 而体力劳动主要依靠男性 女性能够从事的有收入的工作岗位极少

新时代 随着科技的进步 社会上能够提供很多女性岗位 双方都能够拥有一定的经济来源 这是一个很好的开始 在理想婚姻的第一个阶段 独立出双方原先的家庭生活 这是能够办到的 但是 随着时间的推移 生儿养女 是离不开父母的支持与帮助 那这需要男方还是女方呢 这个问题 我想大家都有自己的情况下属于自己的答案 再往后还有赡养的问题

婚姻是很复杂的 但是基于爱情的婚姻是简洁明了的 双方只管付出就是

关于婚前同居 我是支持的 如果两个人决定结婚了 那在婚前最好是要同居 对生活习惯 性都是必要的磨合 如果存在非常难以调和的矛盾 可以不结婚

关于这一块还有很多内容需要补充 先到此吧
