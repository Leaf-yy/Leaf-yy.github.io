# 贝叶斯定理

# 概念

# 1. 条件概率 （后验概率）

事件 A 在另一个事件 B 已经发生的条件下发生的概率，既 在 B 条件下 A 的概率

P(A│B)=P(A∩B)P(B)P(A│B)=\frac{P(A∩B)}{P(B)}P(A│B)=P(B)P(A∩B)​

# 2. 联合概率

两个事件共同发生的概率

P(A,B)P(A,B)P(A,B)

# 3. 边缘概率（先验概率）

某个事件发生的概率。是这样得到的：在联合概率中，把最终结果中那些不需要的事件通过合并它们的全概率，而消去它们（对离散随机变量用求和得全概率，对连续随机变量用积分得全概率），称为边缘化（marginalization），A 的边缘概率：P（A）

# 4. 全概率

设事件 A.... 构成一个完备事件组，既它们两两不相容，和为全集且任一事件概率大于 0

这是 由因推果 的思想

P(B)=∑n=1∞P(An)P(B∣An)P(B)=\sum_{n=1}^{∞}P(A_n)P(B|A_n)P(B)=n=1∑∞​P(An​)P(B∣An​)

# 5. 贝叶斯公式

由果推因

P(Ai∣B)=P(Ai)P(B∣Ai)P(B)=P(Ai)P(B∣Ai)∑n=1∞P(An)P(B∣An)P(A_i|B)=\frac{P(A_i)P(B|A_i)}{P(B)}=\frac{P(A_i)P(B|A_i)}{\sum_{n=1}^{∞}P(A_n)P(B|A_n)}P(Ai​∣B)=P(B)P(Ai​)P(B∣Ai​)​=∑n=1∞​P(An​)P(B∣An​)P(Ai​)P(B∣Ai​)​

# 例子

# 一

假设我喜欢一个女生，现在判断这个女生是否喜欢我。根据平时的交流假定（先验概率）女生喜欢我的概率为 50%。在相处过程中，该女生经常主动勾搭，约饭等 那么

可能性函数：P(B∣Ai)P(B)>1可能性函数：\frac{P(B|A_i)}{P(B)}>1可能性函数：P(B)P(B∣Ai​)​>1

假设该值为 1.5 则

记：

事件A：女生喜欢我事件B：女生经常主动勾搭，约饭等P(A∣B)=P(A)P(B∣A)P(B)=0.5∗1.5=0.75事件A：女生喜欢我\\ 事件B：女生经常主动勾搭，约饭等 \\ P(A|B)=\frac{P(A)P(B|A)}{P(B)}=0.5*1.5=0.75事件A：女生喜欢我事件B：女生经常主动勾搭，约饭等P(A∣B)=P(B)P(A)P(B∣A)​=0.5∗1.5=0.75

女生经常主动勾搭，约饭等 该行为下喜欢我的概率为 0.75。不是 1。

当然这些样本来分析感情是不足的

总结：人生最大的错觉就是 他 / 她 喜欢你 只有数学不会骗你！！！

# 二

假设你们成功牵手，你发现这个妹子经常半夜跑去蹦迪，这个时候你想知道这个妹子是不是渣女，于是有了下面的故事，根据某某某网站数据统计有以下数据：

记：A1: 渣女 A2：正经女 B1: 蹦迪 B2 不蹦迪

0.30.7
0.7（B1|A1）渣女蹦迪（B1|A2）正经女蹦迪0.1
0.3（B2|A1) 渣女不蹦迪(B2|A2) 正经女不蹦迪0.9

P(A1∣B1)=P(A1)P(B∣A1)P(B)=0.3∗0.7/0.28=0.75P(A1)=0.3P(B1∣A1)=0.7P(B1)=∑n=1∞P(An)P(B1∣An)=P(A1)P(B1∣A1)+P(A2)P(B1∣A2)=0.3∗0.7+0.7∗0.1=0.28P(A1|B1)=\frac{P(A1)P(B|A1)}{P(B)}=0.3*0.7/0.28=0.75\\\\ P(A1)=0.3\\\\ P(B1|A1)=0.7\\\\\\ \begin{aligned} P(B1)&=\sum_{n=1}^{∞}P(A_n)P(B1|A_n)\\&= P(A1)P(B1|A1)+P(A2)P(B1|A2) \\&=0.3*0.7+0.7*0.1\\&=0.28\end{aligned}P(A1∣B1)=P(B)P(A1)P(B∣A1)​=0.3∗0.7/0.28=0.75P(A1)=0.3P(B1∣A1)=0.7P(B1)​=n=1∑∞​P(An​)P(B1∣An​)=P(A1)P(B1∣A1)+P(A2)P(B1∣A2)=0.3∗0.7+0.7∗0.1=0.28​

经常半夜出去蹦迪 该前提下 女生是渣女的概率为 0.75

当然样本依旧是不足的

总结：不要跟经常半夜蹦迪的 男生 / 女生 谈恋爱 只有数学不会渣你 跟数学谈

# 生活中的哲学思想

一件事情不去做 那么实现的概率为零，去做了，成功的概率再小都会有，即便是 0.0001，一番又一番努力后概率逐渐提升，你会发现成功无限逼近 100%，依旧可能不会成功，那么就没有必要为付出努力而没有收获而感到伤心了，失败始终存在！

这世间只有成功率为 100% 的事情 做了才会 100% 成功。

比如 拉屎了 你一定吃了东西，

你会怎么选呢
